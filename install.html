Gideros Heyzap Plugin

Android installation

1) Gideros project

  • Create Gideros project
  • Export it as Android project
  • Import it in Eclipse

2) Add HeyzapSDK

  • Download HeyzapSDK: http://developers.heyzap.com/
  • Import it as "Existing Android Code Into Workspace"
  • Add HeyzapSDK to your project (make sure both projects are in the same folders, as for example in one workspace):
    • Right click on your project
    • Select Properties
    • Choose Android
    • In the Library section click Add and select HeyzapSDK
  • More on Heyzap SDK installation here: http://developers.heyzap.com/docs/android_sdk_setup_and_requirements

3) Copying files

  • Copy libs folder to exported project
  • Copy src folder to exported project

4) Modify Android manifest

  • Add permissions: <uses-permission android:name="android.permission.ACCESS_NETWORK_STATE"/>
  • Add receiver to application tag to get credit for installed apps: 
    • <receiver android:name="com.heyzap.sdk.PackageAddedReceiver">
 			<intent-filter>
  				<data android:scheme="package"/>
  				<action android:name="android.intent.action.PACKAGE_ADDED"/>
 			</intent-filter>
		</receiver>

5) Modify Main activity file

  • Load library: System.loadLibrary("heyzap");
  • Add external class: "com.giderosmobile.android.plugins.heyzap.GHeyzap"

IOS Installation

1) Gideros project

  • Create Gideros project
  • Export it as IOS project
  • Open project in Xcode

2) Add HeyzapSDK

  • Download HeyzapSDK: http://developers.heyzap.com/
  • Drag Heyzap.bundle and Heyzap.framework into your project
  • Add AdSupport.framework:
    • In the project navigator, select your project
    • Select your target
    • Select the 'Build Phases' tab
    • Open 'Link Binaries With Libraries' expander
    • Click the '+' button
    • Select AdSupport.framework
    • Click Add
  • More on Heyzap SDK installation here: http://developers.heyzap.com/docs/ios_sdk_setup_and_requirements

3) Copying files

  • Copy Plugins folder into your Xcode project folder
  • Add files to your Xcode project:
    • Right click on Plugins folder in your Xcode project
    • Select Add file to "Your project name"
    • Select heyzap.h, heyzapbinder.cpp and heyzap.mm

More information on Heyzap

Documentation

Require plugin

require "heyzap" -- let Gideros know you will be using heyzap plugin, which would automatically create a heyzap variable

Methods:

  • heyzap:init(useAds, appID) --initialize plugin (bool useAds - will you use ads, string appID - your IOS appID (for crosscompatability can also be passed to Android, won't do any harm))
  • heyzap:showAd(androidType [, androidBannerAlignment]) --display ad (for Android: type interstitial or banner, and banner alignment: top (default value), center and bottom) (for IOS can be called without any value or passing explanatory string for tracking, as "Game Pause ad")
  • heyzap:hideAd() -- dismises advertisement, works only on Android, but can be called also on IOS, just won't do anything
  • heyzap:submitScore(score, displayScore, leaderboardId) -- submit score for specified leaderboard
  • heyzap:showLeaderboard([leaderboardId]) -- display default leaderboard or if leaderboardId provided, display specified leaderboard
  • heyzap:unlockAchievement(achievementId) -- unlock specified achievement
  • heyzap:showAchievements() -- show achievements
  • heyzap:checkin([message]) -- allow user to checkin in your game with provided optional message

Events:

  • Event.AD_RECEIVED --ad is displayed
  • Event.AD_FAILED --ad could not be displayed
  • Event.AD_ACTION_BEGIN --some action on ad begun (user click)
  • Event.AD_DISMISSED --ad is dismissed/removed
